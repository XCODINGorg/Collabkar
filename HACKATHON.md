# Hackathon Checklist

Use this file as your submission / demo checklist.

## Demo Flow

1. Open landing page
2. Click **Sign up** → create a Creator account → land in Creator dashboard
3. Log out → Sign up as Brand → land in Brand dashboard
4. Log in as Admin (`admin` / `1234`) → open both dashboards

## Pitch (30 sec)

- Problem: creators and D2C brands struggle to manage deals across WhatsApp, spreadsheets, and DMs.
- Solution: CollabKar centralizes campaigns, deliverables, payouts, and matching.
- Differentiator: role-based dashboards + AI-assisted pricing guidance (demo).

## What to Improve Next

- Persistent DB (Postgres/Mongo) + migrations
- Real campaign/deal entities and messaging
- Payments/escrow integration
- RBAC + audit logs + admin tooling
- Rate limiting + email verification + password reset
